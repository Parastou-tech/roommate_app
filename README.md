# Roommate App Prototype

This is a Next.js prototype for Prep Pair Roommate App.

## Prototype Pages

- `/register`
- `/login`
- `/home`
- `/meals`
- `/inventory`
- `/chores`
- `/collab`

## Instructions To Run The Prototype

From the repository root:

```bash
cd roommate_app
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.
